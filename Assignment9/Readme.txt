A web application to apply for Jobs. This web application is built using React Js in the front end and Node Js
in the backend. And Mongodb database is used to store the data.

Steps to run the backend server
1) Go to Assignment8 in Terminal
2) Type node server.js
 
Steps to run the frontend
1) Open my-app in Terminal
2) Type npm start

Functionality:
- User needs to login with their credentials to access the application
- Once the user is logged in, they will be able to view the following pages
1. Home - About the application and the services provided by them
2. Jobs - All the available jobs will be shown here and the list of jobs
3. About - About the founders of this application
4. Contact - Contact details to send message
