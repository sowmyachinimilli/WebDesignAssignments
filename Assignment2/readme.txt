<!DOCTYPE> tag is the very first line in the HTML document. It tells the browser what version of HTML the document is written so that the browser knows what to expect. This tag is also commonly referred to as the <!DOCTYPE> element.

The <header> tag represents the header of a document or a section.

The <main> tag represents the main or dominant content of the document.

The <footer> tag represents the footer of a document or a section.

The <nav> tag defines a section of navigation links.

The <ul> tag defines an unordered (bulleted) list.

The <li> tag defines a list item.

The <div> tag defines a division or a section in an HTML document.

The <h1> to <h6> tags are used to define HTML headings.

The <i> tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

The <a> tag defines a hyperlink, which is used to link from one page to another.

The <p> tag defines a paragraph.

The <form> tag is used to create an HTML form for user input.

The <input> tag specifies an input field where the user can enter data.

The <textarea> element is often used in a form, to collect user inputs like comments or reviews.

The <link> tag defines the relationship between the current document and an external resource.

The <figcaption> element to define a caption for the photo.

The <table> HTML element represents tabular data — that is, information presented in a two-dimensional table comprised of rows and columns of cells containing data.
    <th> -  defines a cell as header of a group of table cells. 
    <tr> -  defines a row of cells in a table. The row's cells can then be established using a mix of <td> (data cell) and <th> (header cell) elements.
    <td> -  defines a cell of a table that contains data


Important CSS used:

* float- Elements that seem to “float" on the right or left side of either the browser window or another element are often configured using the CSS float property.

* clear - Useful to “clear” or terminate a float

* overflow - Useful to “clear” or terminate a float before the end of a container element

* list-style-type: none; - Removes the bullets/list style type

* text-decoration: none; - Removes any type of text decoration, example: underline for text in anchor tags

* a:hover - To add CSS specific to the tag when a cursor is hovered upon.

* position: absolute; - Precisely specifies the location of an element outside of normal flow
in in relation to its first parent non-static element
